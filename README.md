# isaaccorbrey.com

My personal website. See it in action: https://www.isaaccorbrey.com

## Contributing

```bash
# Download the repository
git clone https://github.com/icorbrey/isaaccorbrey.com.git

# Install dependencies
npm install

# Start the development server
npm run dev -- --open
```
