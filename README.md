# isaaccorbrey.com

My personal site.

## Contributing

```sh
git clone https://github.com/icorbrey/isaaccorbrey.com.git
cd ./isaaccorbrey.com

# Run Storybook
npm run storybook

# Run the site
npm run dev -- --open

# Preview the production build
npm run preview
```
