The source for my personal site. You can view the live version at
[isaaccorbrey.com](https://isaaccorbrey.com).

## Getting started

Clone the repository and install dependencies:

```sh
git clone https://github.com/icorbrey/isaaccorbrey.com.git
cd isaaccorbrey.com
pnpm install
pnpm run dev
```
